<h1 align='center'>Vite + React + TypeScript + Eslint + Prettier + Vitest + ChakraUI ⚡</h1>

## **Using 📦**

1. Clone Template

```bash
git clone https://github.com/tpaphysics/vite-react-template
```

2. Install Packages

```bash
yarn install
```

3. Start Project

```bash
yarn vite
```

4. If you using git, delete the existing folder .git after cloning (open `git bash` or other terminal)

```bash
rm -rf .git
```

## **Tests**

```bash
yarn test
```

```bash
yarn test:ui
```

```bash
yarn coverage
```

## **Options ✍️**

1. Check lint

```bash
yarn lint
```

2. Fix lint

```bash
yarn lint:fix
```

3. Check prettier

```bash
yarn prettier
```

4. Fix prettier

```bash
yarn prettier:fix
```

5. Fix lint and prettier

```bash
yarn format
```
